# Data Dictionaries

This repo includes data dictionaries formatted according to the RADx data dictionary specification:

https://raw.githubusercontent.com/bmir-radx/radx-data-dictionary-specification/refs/heads/main/radx-data-dictionary-specification.md

## Files
- `dictionaries/up.dd.csv` — RADx-UP data dictionary
- `dictionaries/rad.dd.csv` — RADx-rad data dictionary
- `dictionaries/gcb.dd.csv` — RADx Global Code Book data dictionary
